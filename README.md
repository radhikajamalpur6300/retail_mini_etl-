# Retail Mini ETL

This beginner-friendly Python project provides a simple ETL structure with source code, tests, and data folders.

## Run Steps

1. Create a virtual environment:

   python -m venv .venv

2. Activate the virtual environment:

   - Windows PowerShell:
     .\.venv\Scripts\Activate.ps1

   - Windows CMD:
     .\.venv\Scripts\activate.bat

3. Install dependencies:

   python -m pip install -r requirements.txt

4. Run the project script:

   python -c "import src"

5. Run tests:

   python -m pytest
